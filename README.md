# Driver State Estimation for Adaptive Vehicle Safety Control

## Overview

This project focuses on estimating driver fatigue and attention states using behavioral data such as eye movement, steering input, and vehicle dynamics. The goal is to enhance driving safety by enabling adaptive vehicle control based on real-time driver conditions.

## Key Features

* Driver state estimation using machine learning (Random Forest)
* Integration of multiple behavioral features (eye movement, steering variance, speed variation)
* Potential application to safety systems such as warning alerts and adaptive control

## Motivation

Human factors contribute significantly to traffic accidents. This project aims to bridge human state understanding and vehicle control, contributing to safer and more intelligent mobility systems.
